MoonDrive

Manage all your cloud files in one place.

MoonDrive is a modern cloud file manager that lets you connect and manage multiple cloud storage services from a single app. With MoonDrive, you can browse files, preview documents, upload content, and download files easily across different cloud platforms.

Supported Cloud Storage

MoonDrive allows you to connect and manage files from:

  • ☁️ Google Drive
  • 💠 Microsoft OneDrive
  • 📦 Dropbox

Switch between accounts quickly and access your files without switching between multiple apps.

Connect Multiple Accounts

Securely sign in to your supported cloud storage providers and manage them from one place.

⬆️

Upload Files

MoonDrive integrates with third-party cloud storage providers to manage your files. We communicate directly with these services using their official APIs. We do not store your login credentials or passwords. Authentication is handled securely using OAuth 2.0.

Google Drive

We use the Google Drive API to: List files, Upload files, Download files, and Delete files.

Google user data is accessed only when you grant permission and is transmitted directly between your device and Google's servers. We do not sell or share this data.

Microsoft OneDrive

We use the Microsoft Graph API to manage your OneDrive files.

Dropbox

We use the Dropbox API to manage files stored in your Dropbox account.

2. File Access (Local & Cloud)

MoonDrive requires permission to access device storage in order to function as a file manager.

Permissions may include:

  • READ_EXTERNAL_STORAGE
  • WRITE_EXTERNAL_STORAGE
  • MANAGE_EXTERNAL_STORAGE

These permissions allow the app to: Display files, Copy or move files, Upload files to cloud storage, Download files from cloud storage, and Delete files when requested. Downloaded files are stored locally on your device.
